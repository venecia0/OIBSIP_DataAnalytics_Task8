# Google Play Store Data Analysis

## Project Overview

This project focuses on analyzing data from the Google Play Store to gain insights into mobile applications. The analysis helps understand patterns related to app categories, user ratings, number of installs, and pricing models. Data analytics techniques were used to explore trends in the mobile application marketplace.

## Dataset

**apps.csv**

The dataset contains information about Android applications available on the Play Store, including attributes such as category, rating, number of installs, app size, price, and whether the app is free or paid.

## Objective

The objective of this project is to explore and analyze Play Store data to identify patterns in app popularity, ratings, and pricing strategies.

## Steps Performed

The following steps were carried out in the analysis:

1. Loaded the dataset and explored its structure
2. Cleaned the dataset and converted columns to appropriate data types
3. Analyzed the distribution of apps across different categories
4. Visualized ratings and install counts using graphs
5. Compared trends between free and paid applications
6. Interpreted the results to identify key insights

## Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Outcome

The analysis provided useful insights into the Play Store ecosystem, including popular app categories, rating patterns, and install trends. It also highlighted differences between free and paid applications.

## Conclusion

This project demonstrates how data analytics can be applied to understand trends in the mobile app market. The insights gained from the analysis can help developers and businesses make better decisions regarding app development and marketing strategies.
